ColorPlus.js
============

A painless way to color your console in node.js.
-----------------------------------------------



To Install
==========

`npm install colorplus`


To Use
======

	var cp = require("colorplus");
	console.log("This is", cp.yellow, "Color", cp.red, "Plus", cp.r, "Test");
